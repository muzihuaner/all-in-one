# Docker 镜像

很多朋友安装 Docker 应用的时候出现下载失败问题。想解决这个问题通常有两种办法，第一种是设置代理，比如在主路由或者旁路由上配置代理，可以让拉取 docker 镜像是通过代理加速。第二种是直接使用镜像加速器。

```
镜像加速器
```
